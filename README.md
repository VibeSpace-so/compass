# Compass

Guided curation and decision layer for vibe coders — by [Vibe Space](https://vibespace.so).

Compass helps users understand where they are in the project journey, what to do next, which tool to use, and what tradeoffs they are making.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS v3
- lucide-react
- localStorage for all data persistence
